赋值运算符

语法:变量名=表达式
=号就代表赋值运算符.先计算右边的,再运行左边的.

算术运算符

​ +:加号
​ -:减号
​ *:乘号
​ /:除号 比如:5/2=2
​ %:余数 比如:5%2=1
​ 其他:
​ i++:就是本身加1 i=i+1是一个意思
​ i–:同理
​ 注意:进行整数计算的时候,出现的小数部分,不参与计算

自动类型转换

规则1：如果一个操作数为double型，则整个表达式可提升为double型
规则2：满足自动类型转换的条件两种类型要兼容数值类型（整型和浮点型）互相兼容
目标类型大于源类型 例如：double 型大于 int 型

强制类型转换

语法:（类型名）表达式
不能自动类型转换的就需要强制转换

Scanner使用步骤

​ (1)指定Scanner类路径
​ import java.util.Scanner;
​ (2)键盘输入
​ Scanner input=new Scanner(System.in);
​ (3)输入提示信息
​ System.out.print("请输入姓名:");
​ (4)通过变量获取控制台输入
​ String name=input.next();
​ (5)展示信息
​ System.out.print("姓名:"+name);

选择结构

​ 1.基本if选择结构
​ 语法：if(条件){代码块}
​ if:如果
​ ():需要进行判断的条件,只能是boolean类型
​ {}：条件成立的时候执行的代码
​ 2.复杂条件下的if选择结构
​ 使用逻辑运算符
​ 1.&&:表示并且，必须两个条件同时满足，结果为真
​ 2.||:表示或者，两个条件有一个满足，结果为真
​ 3.!:表示否定，就是把结果反过来
​ 4.运算符优先级顺序：！>算术运算符>比较运算符>&&>||
​ 3.if-else选择结构
​ 语法:if(条件){代码块1}else{代码块2}
​ else:表示否则
​ 当条件满足执行if里面代码块1的内容，条件不满足执行else里面代码块2的内容
​ 4.多重if选择结构
​ 语法:if(条件){代码块1}else if(条件){代码块2}
​ 特点: else if(){}可以有多个
​ else{}可以省略
​ 代码从上向下执行，条件满足以后剩下的代码就不执行了
​ 5.switch选择结构
​ (1)
语法：switch(条件){
​ case 值:
​ 代码块
​ break;
​ default:
​ break;
​ }
​ (2)特点
​ 1.switch选择结构，条件为等值判断
​ 2.case可以有很多个，但是不要忘记写break
​ 3.default顺序可以改变，但是要注意执行顺序，通常写在末尾，也可以省略
​ (3)比较switch和多重if的结构
​ 相同点：都是用来处理多分支结构
​ 不同点：switch只能进行等值判断
​ if一般用于对某个区间进行判断，也可以等值判断

while循环

​ 语法:while(循环条件){循环操作}
​ 特点:先判断,在执行
​ 使用while循环的步骤
​ 1.先声明变量,并且赋予初始值
​ int i=0;
​ 2.判断循环条件
​ while(i<=10000){循环操作}
​ 3.执行循环操作
​ 4.防止死循环
​ i++;
3.比较两个String类型的值是否相等
​ 数字使用两个等号”==”
​ 字符串使用”变量名.equals(相等的内容)”
4.如何进行程序调试
​ 1.分析错误,设置断点
​ 2.启动调试,可以右键选择DeBug As,或者点击爬虫的按钮
​ 3.单步运行,启动调试以后,运行到设置断点的代码会停住,点击F6可以单步运行程序,观察程序运行的过程
​ 4.观察变量
​ 5.发现问题
​ 6.修改代码,重新运行
​ 7.解决问题

do-while循环结构

​ 语法:do{循环操作}while(循环条件);
​ 特点:先执行,再判断,分号必不可少,初始条件不满足至少执行一次
​ 使用步骤:
​ 1.先声明变量
​ int i=0;
​ 2.执行循环操作
​ do{循环操作}while(循环条件);
​ 3.判断循环条件
​ 4.防止死循环

for循环结构

​ 语法和执行顺序
​ for(1.初始化;2.判断条件;4.更新循环变量){
​ 3.循环操作
​ }
​ 执行顺序:1.–>2.–>3.–>4.–>2.–>3.–>4…循环到结束
​ 特点
​ 表达式全部省略,会造成死循环
​ for(;;){
​ //死循环
​ }

关键单词

​ (1)break
​ 控制流程语句,跳出整个循环,通常和条件语句一起使用
​ (2)continue
​ 只能用在循环里,跳过循环中剩余的语句,执行下一次循环
​ (3)return
​ 都能使用,结束程序

如何进行程序调试

     1. 分析错误，设置断点
     2. 启动调试
     3. 单步运行，启动调试以后，运行到设置断点的代码将停住点击F6可以单步运行程序，观察程序运行过程
     4. 观察变量
     5. 发现问题
     6. 修改代码，重新运行
     7. 解决问题

数组

1.什么是数组

​ 数组就是一个变量,存储相同类型的一组数据
​ 声明数组就是在内存开辟出一连串连续的空间

2.数组的基本组成要素

      1. 标识符:数组的名称,用于区分不同的数组
      2. 数组元素:数组当中存放的值
      3. 元素下标:每一个存放进来的值都有一个对应的编号,从0开始,数组中每个元素都可以通过下标来访问
      4. 元素类型:存放值的具体类型,决定了是什么类型的数组

3.使用数组的好处

​ 减少代码量,有利于数据的处理,使用的时候要注意,数组中所有的元素必须属于相同的数据类型

4.如何使用数组

1.声明数组,就是告诉计算机数据类型是什么

语法:数据类型 数组名[]; 或者数据类型[] 数组名;

注意:声明数组的时候括号里不需要规定数组的长度.

2.分配空间,就是告诉计算机分配几个连续的空间

语法:声明数组并同时分配空间

​ 数据类型[] 数组名称=new 数据类型[大小];

注意:分配空间编号是从0开始的,长度固定不变,避免数组下标越界

3.赋值,就是向分配好的空间里存放数据

​

5.Arrays工具类的使用

​ 1.判断数组之间是否相等
​ Arrays.equals(数组a,数组b);显示的结果为boolean
​ 2.排序,对数组进行升序排列
​ Arrays.sort(数组);
​ 3.替换数组元素,把数组换成字符串输出
​ Arrays.toString(数组);
​ 4.改变数组长度
​ 新的数组=Arrays.copyOf(要改变的数组,长度);
