MySQL 使用存储过程批量插入

存储过程：是保存一条或多条 SQL 的批处理脚本

存储过程的优点

  • 存储过程可以把复杂逻辑封装在一个单元中
  • 存储过程可以回传值，并可以接收参数
  • 存储过程可以用在数据检验和业务逻辑实现

数据库为8.0
