学习小网站

理论知识虽然比较干货，但学起来确实比较枯燥。理解能力比较差话还会容易误解，如果搭配一些图解或动画，学起来会容易许多，也会更有趣。

动画学Git

这是个叫做 Learning Git Branching 的项目，强烈推荐。

在项目中用到 Git 时，我们可能会自己发现来来回回就会用 clone，add .，commit，merge，push，pull 这几个命令，其他的命令完全不会。

这个动画网站吧常见的Git命令讲解的非常清楚，配上可视化的动画效果，学习起来事半功倍。

练习正则表达式

练习平台RegexOne

前面有基本教程，后面有一些常见的正则表达式题目，比如判断邮箱、URL、电话号，或者抽取日志的关键信息等等，注意是英文的。

写出符合要求的正则表达式，就可以进入下一个问题，关键是每道题还有标准答案，可以点击下面的 solution 按钮查看。

再说个在线测试工具 RegExr

可以看见，输入对应的文本和正则模式串后，网站会给正则表达式自动添加一些添加好看的样式，自动在文本中搜索模式串，还会高亮显示匹配的字符串，显示每个分组捕获的字符串。

练习SQL

SQLBolt 是一个适合小白学习 SQL 的网站，这里由浅及深的介绍了 SQL 的知识，每一个章节是一组相关的 SQL 知识点，且配备着相应的练习。

基础学了之后，可以尝试挑战一下进阶版： leetcode

而且可以在线测评，和评测算法题一样，也会让你很直观的看到自己所写的 SQL 的运行速度等，同样可以去借鉴别人优化好的 SQL。
